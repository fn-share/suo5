适用于 NBC NAS 的 suo5 客户端管理器
--------------------------------

&nbsp; 

### 适用机型

适用于 NBC NAS 如下机型：

1. 1a, 1b, 1c
2. 2a, 2b, 2c
3. 3a, 3b, 3c

&nbsp;

### 如何安装？

第 1 步: 打开 NBC NAS 设备主页，进入 “管理 | 安装应用” 菜单

第 2 步：将如下链接拷贝到 “安装包 URL” 输入框，然后依次点击 “下载” 与 “安装” 按钮。

``` bash
https://github.com/fn-share/suo5/archive/refs/heads/main.zip
```
